# SkidHub V.1

A black & white, space-themed Roblox script hub — tools for skids and much more.

> This is only **V.1** — more coming soon.

SkidHub gives you a clean, modern menu of scripts you can run with one click. The
whole library is pulled live from a GitHub-hosted `Scripts.json`, so it stays up to
date automatically. Every script is open source except for the actual menu which you can request source by DMing me, my dc is at the bottom of this file, nothing sketchy

## Features

- **One-click run** — hit `RUN` on any card to execute the script.
- **Built-in source viewer** — click a card to read the code (with syntax shading,
  line numbers, and a copy button) before you run it.
- **Auto-updating library** — loads from `Scripts.json` and re-syncs in the
  background, so the list is always current. Hit `↻ sync` to refresh on demand.
- **Search + tabs** — filter modules instantly, and a **Favorites** tab for the
  ones you use most.
- **Slick UI** — animated boot screen, starfield/space theme, draggable + resizable
  window, minimize, and a live status bar (FPS, ping, clock, game + player count).
- **RightShift** toggles the hub on/off.

## Usage

Paste our loadstring, `loadstring(game:HttpGet("https://raw.githubusercontent.com/m1kxCodes/SkidHub/refs/heads/main/main.lua"))()` into your executor and run it for automatic updates, or paste the contents of `main.lua` into your executor and run it for a static build.

> Requires an executor that provides `loadstring` and an HTTP function
> (`game:HttpGet` / `request`). For your own use only.
